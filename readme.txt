forkify is an app that allows the user see differents recipes and the portion of each ingredient, also you can save (temporally) your favorite recipes.

to make it works, you have to:

1. download or clone the repository

2. install node or look at your version

3. be in the project path in the command line.

4. in the command line execute the next command "npm install" it's gonna install the dependencies of the package.json

5. finally to make it works, execute the command: "npm run start", this instructions is defined in package.json and launch a server for the app
